# MiniProj2_LDA_functions

<!-- badges: start -->
<!-- badges: end -->

The goal of MiniProj2_LDA_functions is to provide a comprehensive, high-performance R toolkit for extending Linear Discriminant Analysis (LDA) to robust, sparse, and adversarial settings. This project encompasses several advanced enhancements to classical LDA, integrating projection-based dimensionality reduction, Empirical Likelihood (EL) estimation, LASSO/Elastic-net penalties, and C++ optimizations for high-dimensional and ill-behaved datasets.

## Implementation Details

The core implementations are stored in `main/LDA_alpha.R` and `main/lda_cpp.cpp`. The algorithms developed in this package include:

- **Moment Matching LDA (`lda_moment` & `lda_moment_cpp`)**: An efficient SVD-based linear discriminant boundary estimator that mitigates rank deficiencies. Includes a 32-bit float C++ wrapper using `Rcpp` for extreme performance in scaling contexts.
- **Sparse LDA (`lda_dsda_train` & `lda_dsda_test`)**: Leverages `glmnet` (LASSO/Elastic Net) by reframing classification as a penalized least-squares problem using response label encodings. Followed by optimal threshold intercept derivation.
- **Adversarial & Fair Feature Selection LDA (`lda_fair`)**: Implements FAIR feature selection criteria. Extracts dimensions that truly distinguish classes against the maximum eigenvalue directions of pooled covariances, mitigating adversarially deceptive features.
- **SVD/PCA LDA Projections (`lda_2rs` & `lda_2rs_moment`)**: Solves extreme High-Dimension Low-Sample-Size (HDLSS) setups by projecting the dataset into a lower $n-2$ subspace, retaining critical discriminant energy.
- **Empirical Likelihood LDA (`lda_rs_el`)**: Fuses dimensionality reduction with Empirical Likelihood (via the `gmm::gel` backend). Replaces Gaussian data assumptions with data-driven non-parametric constraints, generating boundaries highly robust against heavy-tailed outliers or misspecified distributions.
- **Monte Carlo Bayes Error Estimator (`lda_bayes_error_rate`)**: Provides theoretical optimum baselines.

## Testing and Simulation Modules

A comprehensive suite of simulation and benchmarking scripts is available to evaluate the methodologies mathematically derived in the `mini-proj-2.pdf` report:

1. **`test_of_extreme_dimension_Sec2-3.R`**: Scaling validation showcasing computation stability and execution speed against extremely large parameters $p$.
2. **`test_of_lda_moment_Sec2-2.R`**: Benchmarks `lda_moment` correctness via simulated baseline comparisons to exact properties of `MASS::lda`.
3. **`test_of_real_data_examples_Sec3-2.R`**: Demonstrates the real-world utility of the classifier pipelines.
4. **`test_of_RS-EL_SpikeCov_Sec3-3-1.R`**: Parallelized simulation targeting structural challenges posed by spiked block covariance matrices.
5. **`test_of_RS-EL_MisSpecified_Sec3-3-2.R`**: Tests algorithm resilience when assumptions drastically fail due to unmeasured cross-sectional sub-group mixtures (producing misaligned means across latent traits).
6. **`test_of_Adversarial_data_case_Sec4.R`**: Explores adversarial extreme-subspace hiding architectures and measures performance degradation. Computes Monte-Carlo expected Bayes Error boundaries prior to empirical evaluations.

## Accompanying Report

Theoretical mathematical derivations, analytical properties of the optimal score equations, and detailed discussion of the computational outcomes generated by the test scripts are contained within the `mini-proj-2.pdf` draft report.

## Setup and Usage

Please ensure the necessary upstream dependencies are installed:
```R
install.packages(c("MASS", "glmnet", "gmm", "Rcpp", "float", "RhpcBLASctl", "ggplot2", "tidyr", "doParallel"))
```

Load the core algorithms:
```R
# Directly sources the backend layout and compiles lda_cpp.cpp into memory
source("main/LDA_alpha.R")
```
